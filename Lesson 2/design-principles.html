Repetition

www.microsoft.com

Repetiiton in web design and in design in general means that you can find elements in design that are repetitive which gives to the design harmony and balance. Microsoft used repetition when using the same blue for background colors and links. It also uses repetition with the font. It is using the same font in the whole page. Another example of repetition is found is in the icons under the firs image. They are of the same style and color.

Rule of Thirds

Microsoft

www.microsoft.com

Rule of thirds consists on dividing and element in 3x3 sections and putting the elemetns inside it in certain position that have been proved to be better accepted to the sight. For example, Microsoft uses the rule of thirds in the first image when in th image, the main caracters are on the upper-right are and the content on the upper-left are. It creates balance and goes along with the rule of thirds.
